# Introduction

FoodPanda is a food delivery company in Bulgaria similar to Deliveroo. 
The idea of this project is that it displays the recipes for the food delivered by the company.
Users can pick a given food and recieve all the ingredients used to make it.
It is made using food2fork API.

# Functionality
- Search for a recipe by a given word
- Select a recipe to get information about it:
  - Ingredients
  - Number of servings
  - Minutes to make
- Number of servings can be adjusted and then the number of ingredients changes
- Recipe can be added to shopping list
- In the shopping list the count of a given ingredient can be adjusted
- Recipe can be liked and added to liked recipes list
- Liked recipes list is saved and can be seen after reloading the page
- Simple functions as
  - Removing a recipe from liked list
  - Removing a given ingredient from shopping list
  - Heart is hidden when there are no liked recipes
  - Pagination
  - More...
  
# Made Using:
- HTML
- CSS
- JavaScript
- API - food2fork
- API - LocalStorage
- Babel
- Webpack
